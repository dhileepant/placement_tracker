# Placement Tracker Dashboard

The Placement Tracker Dashboard is a web-based application designed to help students and administrators track placement progress and visualize key placement metrics. It provides a centralized platform to manage placement data and monitor progress efficiently.

## Overview

This project simplifies placement tracking by offering structured dashboards and visual insights. It supports role-based views for students and administrators and presents placement information using interactive charts and progress indicators.

## Features

### Student Dashboard
- View individual placement progress
- Track applied companies and selection stages
- Monitor placement status visually

### Admin Dashboard
- Manage student placement data
- View overall placement statistics
- Analyze placement performance using charts

### General Features
- Role-based authentication
- Interactive charts and progress tracking
- Responsive and modern UI

## Tech Stack

### Frontend
